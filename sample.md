## Mermaid Diagrams

### Flowchart

```mermaid
flowchart TD
    A[Open Markdown] -->|:Md| B[Render HTML]
    B --> C{S3 configured?}
    C -->|Yes| D[Upload to S3]
    C -->|No| E[Write local file]
    D --> F[Print URL]
    E --> F
```

### Sequence Diagram

```mermaid
sequenceDiagram
    participant N as Neovim
    participant R as Rust Binary
    participant S as S3 Storage

    N->>R: stdin: markdown content
    R->>R: comrak parse
    R->>R: mermaid render
    R->>R: syntect highlight
    alt S3 configured
        R->>S: PUT object
        S-->>R: 200 OK
        R-->>N: stdout: URL
    else Local mode
        R->>R: write file
        R-->>N: stdout: path
    end
```
